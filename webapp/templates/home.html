Bem-vindo ao Classifier

Esta é uma aplicação de classificação de instâncias que permite classificar dados usando diferentes modelos.

Como funciona:

Você pode carregar um arquivo CSV contendo instâncias a serem classificadas. Em seguida, selecione um modelo disponível para classificação.

A aplicação processará os dados usando o modelo selecionado e exibirá os resultados.
